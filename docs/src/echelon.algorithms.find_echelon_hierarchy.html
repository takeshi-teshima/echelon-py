echelon.algorithms.find_echelon_hierarchy module

find_echelon_hierarchy(peak_echelons, foundation_echelons, oracle)[source]
Parameters

  • peak_echelons (EchelonsType) –

  • foundation_echelons (EchelonsType) –

Returns

Root node of the hierarchy.

Return type

anytree.node.node.Node

Notes

Algorithm

The algorithm is recursive.

  1. Given a list of echelon indices, find the maximum index value, and consider it the root node \(R\).

    1. If the rest of the indices is empty, return.

    2. Otherwise, split the rest of the indices into connected components (where echelons are considered to be adjacent when they have adjacent elements).

      • Parse each connected component by calling this procedure recursively.

      • For each of the connected components, we obtain its root node \(r\). Attach \(r\) to \(R\) as its child.

Examples

>>> from echelon.test import AdjacencyOracleMockup, _visualize_echelon_hierarchy
>>> import string
>>> oracle = AdjacencyOracleMockup(list(string.ascii_uppercase)[:25])
>>> peak_echelons = [['Q', 'P', 'R'], ['N'], ['G', 'F', 'H'], ['D'], ['X']]
>>> foundation_echelons = [['M', 'O', 'S', 'L', 'T', 'K', 'U'], ['C', 'E', 'I'], ['B', 'J', 'V'], ['A', 'W', 'Y']]
>>> _visualize_echelon_hierarchy(find_echelon_hierarchy(peak_echelons, foundation_echelons, oracle))
9
├── 8
│   ├── 7
│   │   ├── 4
│   │   └── 3
│   └── 6
│       ├── 2
│       └── 1
└── 5
